# **App Name**: Gym Buffet Manager

## Core Features:

- Product Management: Manage products: add, edit, delete products with details like name, stock, and purchase price.
- Ingredient Management: Manage ingredients: add, edit, delete ingredients with details like name, stock, and purchase price.
- Food Recipe Management: Define food recipes by combining ingredients and quantities. Calculate food cost based on ingredient prices.
- Order Management: Create and manage orders containing products and foods. Track customer orders and payment methods.
- Inventory Tracking: Track inventory levels for both products and ingredients. Automatically adjust stock based on orders, purchases, and waste.
- Accounting: Calculates costs and profit for both products and food. Provides data such as stock value, wasted food value, transport costs.
- Generate Usage Reports: Leverage a generative AI tool that analyzes historical data, predicting food usage to reduce waste. Generates reports on the application state.  

## Style Guidelines:

- Primary color: Deep, earthy green (#4CAF50) to represent freshness and health.
- Background color: Light grey (#F5F5F5), providing a neutral backdrop.
- Accent color: Golden yellow (#FFC107), evoking images of grains, breads, cereals etc.
- Body and headline font: 'PT Sans', sans-serif, blends approachability and a modern feel. 
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean, modern icons representing food, ingredients, and inventory management actions.
- Grid layout for Products/Foods on Order Screen to ensure one-hand usability. Cart always visible at the bottom of the Order Screen.
- Subtle transitions when adding or removing items from the order.
